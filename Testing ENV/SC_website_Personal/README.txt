SETTING UP
=============
Here's a rough guide on how to set up the assignment.

1. Start up MySQL Workbench. Import the sql file to create the necessary schema.
2. Navigate to `config.js`. You are to change the code to match your database credentials.
3. Install the relevant packages. This can be done by running `npm install` in your project directory.
4. Run the project with `npm run start` or `npm run dev` (hot-reload enabled).
